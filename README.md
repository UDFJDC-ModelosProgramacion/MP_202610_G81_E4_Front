## Descripción del Proyecto

Este proyecto corresponde al desarrollo de una aplicación web dividida en dos capas principales:

- **Backend** desarrollado con **Spring Boot**

  https://github.com/UDFJDC-ModelosProgramacion/MP_202610_G81_E4_Back.git
- **Frontend** desarrollado con **React + Vite**
  
El sistema sigue una arquitectura cliente-servidor, donde el frontend consume los servicios REST expuestos por el backend.

---
# ¿Como ejecutar el FrontEnd?

##  Requisitos Previos

Antes de iniciar, asegúrate de tener instalado:

- **Node.js** versión 20 o superior  

---

# Descargar el Proyecto

##  Opción 1: Descargar con Git (Recomendado)

### 🔹 Backend

Abre una terminal y ejecuta:

git clone https://github.com/UDFJDC-ModelosProgramacion/MP_202610_G81_E4_Back.git

### 🔹 Frontend

Abre una terminal y ejecuta el siguiente comando:

git clone https://github.com/UDFJDC-ModelosProgramacion/MP_202610_G81_E4_Front.git

# Opción 2: Descargar Proyecto en ZIP desde GitHub

Si no tienes **Git** instalado, puedes descargar los proyectos manualmente en formato `.zip`.


# 🔹 Descargar Backend

### Pasos:

1. Ingresa al repositorio:

https://github.com/UDFJDC-ModelosProgramacion/MP_202610_G81_E4_Back

4. Espera la descarga.

5. Extrae el archivo `.zip` en tu computador.

# 🔹 Descargar Frontend

#  Ejecutar el programa

# 🔹 Backend

1. Ingresa a la carpeta del Backend y en la terminal ejecutar la siguiente linea:

   cd MP_202610_G81_E4_Back

2.Ejecutar el proyecto con la siguiente linea:

   mvn spring-boot:run

3. URL Backend:

   http://localhost:8080

# 🔹 Frontend

1. Ingresa a la carpeta del Backend y en la terminal ejecutar la siguiente linea:

   cd MP_202610_G81_E4_Front

2. Instalar dependencias

   npm install

3. Ejecutar el proyecto

   npm run dev

4. URL FrontEnd:

   http://localhost:5173
   
### Importante:

Ejecuta primero el backend.

Luego ejecuta el frontend.

Mantén ambas terminales abiertas mientras trabajas en el proyecto.
